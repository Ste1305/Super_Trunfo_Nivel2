Programa que realiza o cadastro de duas cartas seguindo o modelo do jogo "Super Trunfo", edição Paises, e que realiza comparações entre as cartas decidindo quais valores vencem.

Após a declaração das variaveis, o programa pede ao usuario de cadastrar as informações de duas cartas: Codigo do estado, nome de Cidade, população, area, PIB, pontos turisticos, densidade populacional e PIB per capita;
somando os valores pode-se obtenir o valor Super Poder.

Todos os valores das cartas são visualizados na tela.

Chega-se no menu principal onde pode-se escolher de ver as regras (o valor maior entre o mesmo parametro das duas cartas vence, exceto adensidade populacional, one o menor valor vence), de jogar ou de sair.

Na aba "Jogar!" do menu se escolhem dois parametros para a batalha, e é decretada a carta vencedora.
Existe a possibilidade de fazer uma rodada extra, escolhendo outros dois parametros para outra batalha.
Os parametros devem ser sempre diferentes, isto é, não se pode selecionar duas veezes o mesmo parametro das cartas.

Esse programa utiliza estruturas condicionais tais como if, else if; operador ternario; estrutura switch; manipulaçoes de dados atraves operações aritmeticas, input de dados.
Não são utilizados loops (ainda).
O programa faz parte dos exercicios propostos nas aulas do Tecnólogo "Analise e Desenvolvimento de Sistemas".
